# barkatsonne.com

This is a very simple, single-page website that is actually my first webdev project.
Commissioned by Sonneflower, the purpose is to allow Twitch chatters to select an ElevenLabs voice model,
preview some of the possible audio for that voice, and enter text to be read aloud by that model when
submitted in Twitch chat.

All images and preview audio files are locally hosted, so the page can take a second to load everything, but
it means there is no risk of broken links or security breaches.
